## Phase 2A: Forcing Questions（逼迫性问题）

### Q1: 谁有这个问题？你怎么知道的？

**回答**:
> 现在在各个短视频平台上，都已经出现了一些 AI 动画内容。包括像 YouTube 上面，也有一些直接制作动画并上传的账号，能获得比较不错的流量。但是现在我还没有找到满意的工具，不知道他们具体是用什么做的。我粗略调研了一下，感觉目前还没有一个能够一统天下的产品。所以我的猜测是，现在有了各式各样的小产品，用户可能还是在手动去做这些事儿。

**追问**: 这是基于直接用户对话还是观察？
**回答**: 只是观察。

**评估**: 需求信号来自市场观察而非用户访谈，基础较弱但方向合理。

### Q2: 他们现在用什么？为什么很糟糕？

**回答**: 不确定。

**评估**: 这是最大的知识缺口。不知道用户当前的替代方案，就很难设计出更好的产品。

### Q3: 你为什么会赢？

**回答**: 不确定。希望通过调研来找到答案。

**评估**: 尚未形成竞争优势假设，需要通过市场调研和产品差异化来建立。

## Phase 2.75: Landscape Research（市场调研）

### 竞品调研结论

详见 `competitors.md`，核心发现：

1. **市场早期但不空** — 已有 10+ 产品在位，但尚无绝对赢家
2. **两种主要路径**: 端到端管线（MkAnime）vs 单功能工具（DomoAI）
3. **大流量玩家正在进入** — Raphael (3M MAU) 已上线 AI Video Generator
4. **技术壁垒在降低** — 底层模型开源化，工程包装成为关键
5. **成本结构是硬约束** — 动画推理成本远高于图片
6. **9:16 竖屏 = 社交媒体原生** — MkAnime 已经抓住了这个趋势
7. **角色一致性是核心痛点** — 几乎所有端到端产品都在强调这个

### 定价调研结论

详见 `competitors.md` 定价总览表，核心发现：

- 主流价格带: 入门 $10-15/月，中端 $28-40/月，专业 $70-125/月
- 几乎全部 Credits 制（按用量而非按人收费）
- AnimateAI.Pro 定价偏高（$39.99 起步），说明有用户愿意为动画工作流付这个价
- 免费版是获客标配

## Phase 3.5: Codex Second Opinion（第二意见）

使用 Codex CLI (GPT-5.4) 进行独立评估，提出四个任务：

### 1. STEELMAN（最强支持论证）

- 唯一能做到 $10M ARR 的路径：做**发布循环动画竖屏视频的团队的工作流软件**（注：此结论基于 VC 级别目标）
- 核心 wedge 不是生成质量，而是**角色一致性 + 可编辑性 + 产出效率**
- 路径：300-500 个客户，$1.5k-$3k MRR

### 2. ONE REVEALING THING（最关键未知）

- **谁是第一个有循环预算和每周生产习惯的买家？**
- 方法：采访 15-20 个 YouTube Shorts/TikTok 动画频道运营者或小型 agency

### 3. ONE WRONG PREMISE（最可能错误的前提）

- **前提 3 最可能错误："这个领域存在可行的 SaaS 商业模式"**
- 原因：推理成本高、产出不稳定、创作者预算弱、使用模式是脉冲式
- 可能是服务型生意而非纯 SaaS

### 4. 48-HOUR PROTOTYPE（48小时验证）

- 不要写代码，做 concierge test
- 触达 30-50 个创作者/agency，定价 $99-$199
- 信号：48 小时内 3 个认真通话、2 人提交真实素材、1 人付款

### Codex 总结

> "Until you can answer 'who pays, how often, for which painful step?' in one sentence, you should not write product code."

## 创始人的差异化思路

**核心差异化**: 分层编辑能力（非一次性梭哈生成）

- 参考 ChatCut.io 的编辑哲学
- 现有竞品（MkAnime 等）是一次性生成完整视频，修改不灵活
- 提议：AI 分层生成（背景/角色/动作/配音/字幕），用户可单独修改任意层

**实现路径**:
- 路径 A（MVP）：分镜级编辑 — 每个分镜独立生成/重新生成
- 路径 B（愿景）：图层级编辑 — 类似 AE/Figma 的图层概念，AI 驱动

**定价策略**: 高定价，follow gstack 的 boil the ocean 思想

**技术路线**: 调用第三方 API（Kling / Wan / MiniMax）

详见 `open-source-research.md`，核心结论：

- Phase 3 (进阶): + 自托管 AnimateDiff + ComfyUI + IP-Adapter 角色一致性

## Phase 4: Alternatives Generation（方案选择）

### 提出的三个路径

| 路径 | 描述 | 判断 |
|------|------|------|
| A. 分镜编辑器 | 以分镜为核心，每个分镜独立可编辑/重新生成 | ✅ 选择 |
| B. AI 动画 + 编辑器二合一 | 路径 A + 完整时间线编辑器（类似 ChatCut） | ❌ 理论上好但一人团队太费劲 |
| C. 垂直场景专精 | 只做某个垂直领域（如儿童教育动画） | ❌ 不适合 |

### 选择路径 A 的深入讨论

**关于竞品差异化**:
- 创始人判断：不需要过度纠结 MkAnime 的已有功能。市场足够大，所有人都刚起步，容得下多个产品。

**关于角色一致性（核心技术问题）**:
- 真正的痛点可能不只是"能不能改"，而是"改了还能不能保持一致"
- 创始人提出关键技术方向：**基于图的输入的可控视频生成**（Image-to-Video with Reference Control）
- MVP 技术方案：用户上传角色设定图 → 存为"角色资产" → 每个分镜生成时角色参考图自动注入 → 纯 API 调用，不需要自建模型

**关于高定价策略（Boil the Ocean）**:
- 竞品专业版价格带：$76-$129.9/月，市场已验证用户愿意付这个价
- 高价值组合：
  1. 角色资产库（创建一次，跨项目复用，锁定效应）
  2. 分镜级控制 + 重新生成（改了角色还一致）
  3. 系列化/批量生产（跨集复用设定，对 YouTube 频道是杀手级功能）
- 定位不是"AI 视频生成器"，而是"AI 动画内容工作台"

### MkAnime 定价确认（从截图获取）

- Standard: $19.9/月（2,300 credits，~5 videos）
- Pro: $49.9/月（5,900 credits，~13 videos）
- Ultra: $129.9/月（15,500 credits，~34 videos）
- 底层模型: nano-banana-2, Kling/Sora/Wan/Hailuo/Seedance

### AnimateAI.Pro 定价确认

- Standard: $5.99 首周试用 → $39.99/月（500 credits）
- Pro: $99.99/月（2000 credits, Kling 2.0 Master, 最多 10 scenes/video）

### 关键发现

两家竞品底层调用的是同一批模型 API（Kling、Wan、Hailuo 等），技术壁垒在工程包装层而非模型层。

### MVP 定义（已确认）

```
剧本输入 → AI 自动拆 5-8 个分镜 → 每个分镜独立生成预览
→ 用户可以：修改分镜描述重新生成 / 调整分镜顺序 / 删除/增加分镜
→ 全局角色设定锁定（基于图的输入）
→ 一键合成导出
```
