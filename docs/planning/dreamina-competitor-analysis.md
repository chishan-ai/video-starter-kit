## 1. 产品定位

**Dreamina (即梦AI)** 是字节跳动旗下的一站式 AI 创作平台，国际版品牌为 Dreamina（dreamina.capcut.com），国内版品牌为即梦AI（jimeng.jianying.com）。

### 核心定位
- **产品口号**: "All-in-one AI creative suite"（一站式 AI 创作套件）
- **产品形态**: Web 平台 + 移动端 App + CapCut 内嵌集成
- **目标市场**: 全球创作者市场，首批覆盖东南亚、南美、中东、非洲

### 目标用户群
| 用户类型 | 使用场景 |
|---------|---------|
| 短视频创作者 | TikTok/YouTube 内容生产 |
| 数字艺术家 | 概念设计、艺术创作 |
| 游戏开发者 | 角色设计、资产原型 |
| 电商运营 | 产品图片、营销素材 |
| 营销人员 | 海报、广告、社媒素材 |
| 角色设计师 | 2D/3D 角色创作 |
| 教育/自媒体 | 教程视频、科普内容 |

### 与 Mozoria 的用户重叠
Dreamina 的短视频创作者和 AI 动画创作者群体与 Mozoria 的目标用户高度重叠。但 Dreamina 是通用型创作工具，Mozoria 聚焦于分镜级编辑工作流。

## 2. 核心功能

### 2.1 AI 视频生成

| 功能 | 说明 |
|-----|------|
| 文生视频 (Text-to-Video) | 文字描述直接生成视频片段 |
| 图生视频 (Image-to-Video) | 静态图片转动态视频 |
| Seedance 2.0 | 最新旗舰视频模型，支持多模态输入 |
| Photo Dance | 静态图片生成舞蹈动态内容 |
| Sketch to Video | 手绘稿动画化 |
| 文字动画 | 生成平滑的文字动画效果 |
| Add 3s 扩展 | 智能续写视频 +3秒，保持视觉连续性 |

**Seedance 2.0 关键参数**:
- 最长支持 15 秒视频
- 支持 6 种宽高比
- 原生 2K 分辨率输出
- 音视频同步生成（对白、唇形同步、环境音）
- 支持英语、中文、粤语多语言
- 支持文字+图片+视频+音频四种模态混合输入
- 最多同时输入 9 张图片、3 段视频、3 段音频

### 2.2 AI 图像生成

| 功能 | 说明 |
|-----|------|
| 文生图 (Text-to-Image) | 文字描述生成图片 |
| 图生图 (Image-to-Image) | 基于参考图变换风格/内容 |
| AI 艺术生成 | 原创艺术作品 |
| 风格迁移 | 将指定艺术风格应用到照片 |
| AI 超分辨率 | 图像分辨率增强 |

**Seedream 5.0 图像模型**:
- 知识推理与智能编辑能力

## 4. AI 模型与技术

### 4.1 自研模型矩阵

| 模型 | 类型 | 关键能力 |
|-----|------|---------|
| **Seedance 2.0** | 视频生成 | 统一多模态音视频联合生成架构，支持文/图/音/视四模态输入 |
| **Seedream 5.0** | 图像生成 | 知识推理、智能编辑、实时联网检索、多轮图文编辑 |
| **Seedream 3.1** | 图像生成 | 基础文生图/图生图 |

### 4.2 技术特点

- **全部自研**: 字节跳动 Seed 团队自主研发，非第三方模型
- **多模态架构**: Seedance 2.0 采用统一多模态音视频联合生成架构
- **音视频同步**: 视频生成时同步生成环境音、对白、唇形动画
- **物理理解**: 对运动、光影、纹理有较好的物理模拟能力
- **Prompt 遵从度**: 在多项测评中 prompt adherence 评分较高
- **安全措施**: C2PA 水印、AI 内容标识、真人脸部限制、IP 过滤

### 4.3 与竞品技术对比

| 维度 | Seedance 2.0 | Sora 2 (OpenAI) | Kling 3.0 (快手) | Veo 3.1 (Google) |
|------|-------------|-----------------|-----------------|------------------|
| 最大分辨率 | 原生 2K | 1080p | 1080p | 4K |
| 最长时长 | 15秒 | 5-25秒 | 10秒 | 8秒 |
| 音频生成 | 内置音视频同步 | 有限 | 无 | Veo 3有 |
| 多模态输入 | 文/图/音/视 | 文/图 | 文/图 | 文/图 |
| 单片成本 | ~$0.60/10秒 | ~$1.00/10秒 | 更低 | API定价 |
| 优势领域 | 音乐MV、电商、多镜头 | 物理模拟、人物逼真 | 人体动作、性价比 | 分辨率、视觉质量 |

## 5. 与 CapCut/字节跳动生态的关系

### 5.1 生态架构

```
字节跳动 (ByteDance)
├── TikTok (短视频分发平台)
├── 抖音 (国内短视频平台)
├── CapCut / 剪映 (视频编辑工具)
│   ├── Dreamina (AI 创作平台 - 国际)
│   ├── 即梦AI (AI 创作平台 - 国内)
│   ├── Video Studio (画布式AI制作工作区)
│   └── Pippit (专业营销平台)
├── Seed 团队 (AI 模型研发)
│   ├── Seedance 系列 (视频生成模型)
│   └── Seedream 系列 (图像生成模型)
└── BytePlus (企业级 API 服务)
```

### 5.2 整合方式

1. **CapCut 内嵌**: Seedance 2.0 直接集成到 CapCut 编辑器的 AI Video 工具中
2. **Video Studio**: CapCut Web 推出的画布式 AI 制作空间，原生支持 Dreamina 模型
3. **独立平台**: Dreamina 作为独立 Web 站点和 App 运行
4. **Pippit**: 面向专业营销的独立产品线
5. **API 服务**: 通过 BytePlus 提供企业级 API 接入

### 5.3 生态优势

- **流量闭环**: TikTok/抖音 → CapCut 编辑 → Dreamina 创作 → TikTok/抖音 发布
- **用户基数**: CapCut 全球用户超过数亿，直接导流
- **数据飞轮**: 海量用户数据反哺模型训练
- **一站式体验**: 从创意到发布无需跳转第三方工具

## 9. 对 Mozoria 的竞争启示

### 9.1 直接威胁评估: 中等

Dreamina 是通用型 AI 创作工具，功能面广但不深。Mozoria 的分镜级编辑工作流和角色一致性是明确的差异化方向。

### 9.2 需要关注的点

1. **价格压力**: Dreamina 的 $0.60/视频片段对 fal.ai 按量计费模式形成价格参考
2. **音视频同步**: Seedance 2.0 的音视频同步生成是重要能力，Mozoria 的 TTS 方案需要追上
3. **CapCut 生态**: 如果创作者习惯在 CapCut 内完成全部工作流，可能减少对专业工具的需求
4. **角色一致性**: Dreamina 的角色一致性控制被用户评为不足 -- 这正是 Mozoria 的核心价值

### 9.3 Mozoria 的差异化机会

| 维度 | Dreamina 弱点 | Mozoria 机会 |
|------|-------------|-------------|
| 角色一致性 | 缺乏精细控制 | 角色系统 + 参考图 + AI 特征提取 |
| 分镜工作流 | 无分镜级编辑概念 | 分镜级叙事编辑是核心卖点 |
| 叙事连贯性 | 单片段生成为主 | 多镜头故事板 + 连贯叙事 |
| 专业导出 | 通用导出 | Remotion 视频合成 + 专业时间轴 |
| 剧本驱动 | 无剧本概念 | AI 剧本拆分 → 自动分镜 |

### 9.4 建议策略

- **不要试图在通用功能上竞争**: Dreamina 背靠字节的资源和用户量，通用功能竞争无意义
- **深耕垂直工作流**: 分镜叙事 + 角色一致性 + 剧本驱动是 Mozoria 的护城河
- **考虑 Seedance 2.0 作为模型选项**: 如果 Seedance API 价格有优势，可以考虑接入作为视频生成模型之一
- **关注 CapCut Video Studio**: 这是最接近 Mozoria 产品形态的功能，需要持续监控其发展方向
